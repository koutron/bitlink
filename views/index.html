Bitlink

This URL shortener clone takes a URL as a parameter and returns a short code. Once generated, this code (along with the intial bitlink.glitch.me URL) can be used to reroute to the initial URL provided.

Example:
URL = https://www.reddit.com/r/WaterGuy12/comments/6tmcif/commence_the_first_raid_upon_rfireguy12/?st=j6cfo0ce&sh=998cd4cd
Bitlink URL = https://bitlink.glitch.me/877676
